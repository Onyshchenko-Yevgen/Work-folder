1 - Драконья стая

Напишите приложение по генерации драконьей стаи. Пользователь вводит количество драконов в стае. Сгенерировать массив объектов со случайными драконами.

  • Имя дракона
  • Локация, откуда прилетел дракон
  • Пол дракона(М/Ж)
  • Цвет дракона(7 цветов)
  • Возраст дракона(100 до 3000)
  • Головы дракона(1-20)

Входные данные:

Число - целое число от 1 до 100. Количество драконо в стае.

Выходные данные:

Массив объектов(стая). Суммарная сумма голов дракона, мощь стаи драконов, количество драконов мужского пола и количество драконов женского пола. Информацию по самому молодоу дракону и информацию по самому старому дракону. Среднее количество голов у драконов.

3 - Опять Саймон.

Саймон написал код. Саймон уверен, что код в порядке. Но код не работает. Исправь ошибки, допущенные Саймоном.

Входные данные:

Число - количество символов в генерируемой строке

Выходные данные:

Текст - Случайная строка

Примеры:

Развернуть
# IN OUT
1 12 "Random string with length 12 is: cZPhX98!xcV8"
2 1 "Random string with length 1 is: ("
3 "Хммм" "Random string with length NaN is: "

Код Саймона:

Развернуть код 
let organs = 'abcdefghijklmnopqrstuvwxyz0123456789_-!@#$%^&*()',
		orgLen = organs.len,
		len = parseInt(prompt('Enter a len of string')),
		answ = '',
		char = '';

for(let i = 0; i < len; i++){
	char = organs[Math.floor(Math.random() * orgLen)];
	if(Math.random() > 0,5){
		char = char.toLowerCase();
	}
	answ += char;
}

console.log("Random string with length " + len + " is: " + answ);
